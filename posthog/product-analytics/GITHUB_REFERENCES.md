# PostHog GitHub Reference Repositories

Here are existing GitHub repositories with PostHog + Next.js examples you can reference:

## Official PostHog Repositories

### 1. PostHog Identify Demo
**Repository:** https://github.com/PostHog/posthog-identify-demo

**Description:**
- Official PostHog demo for Next.js
- Focuses on user identification, groups, and custom events
- Simple UI to test PostHog features
- Modern React environment

**What it includes:**
- User identification examples
- Custom event tracking
- Group tracking
- Next.js integration

**Useful for:**
- Learning how to identify users
- Understanding event tracking patterns
- Reference implementation
