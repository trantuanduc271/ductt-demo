# Dataproc Template Agent

AI-powered agent for managing and orchestrating Google Cloud Dataproc templates using Google's Agent Development Kit (ADK).

## Tools Overview

| Tool | Purpose |
|------|---------|
| [`get_dataproc_template`](tools/dataproc_template_tools.py) | Retrieve a pre-defined Dataproc template. |
| [`run_dataproc_template`](tools/dataproc_template_tools.py) | Execute a job using a Dataproc template. |
| [`get_transformation_sql`](tools/dataproc_template_tools.py) | Get the transformation SQL from a Dataproc template. |
