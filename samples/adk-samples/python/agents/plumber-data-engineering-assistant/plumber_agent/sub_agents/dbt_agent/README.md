# dbt Agent

AI-powered agent for managing and orchestrating dbt projects using Google's Agent Development Kit (ADK).

## Tools Overview

| Tool | Purpose |
|------|---------|
| [`generate_dbt_model_sql`](tools/dbt_model_sql_generator.py) | Generate SQL for a dbt model. |
| [`deploy_dbt_project`](tools/dbt_project_deployment.py) | Deploy a dbt project. |
| [`run_dbt_project`](tools/dbt_project_runner.py) | Run a dbt project. |
